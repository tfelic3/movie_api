Documentation

This application will allow users to access information about different movies and genres of their liking.

Business Logic URL HTTP Method Request Body Format Response Body Format
Get a list of movies /movies GET None JSON Object will return list of movies.
Get a list of Directors /directors GET None JSON Object will return list of directors.
Get a list of Genres /movies/genres GET None JSON Object will return list of genres.
Allows new users to register /users POST JSON Object holding info about user JSON Object holding info about user that was added
Allows users to update information /movies/users/:Username PUT None Message indicating updated information
Allows users to add a movie to their list of favs /users POST None Returns message about added movies
Allows users to delete movies /users/:Username/Movies/:MovieID DELETE None Removes movie from fav list
Allows users to deregister /users DELETE None Message text indicating deregister
